    如何编辑Unity项目

    BoxHead Survivor 使用Unity开发，同时使用Unity官方提供的版本管理系统（Unity Cloud）的免费额度。 以下的参考内容均为筛选过的，足够精简的内容，最好先看一遍，再上手开发。

    C#语言速成参考资料

    必看基础教程：

    • Hello World - C# 交互式教程简介
    • Coding in C# in Unity for beginners（很短）
    • Scripting in Unity for experienced programmers（很短，跳过DOTS）

    日常开发遇到语言相关问题，尽量参考微软的C#官方文档：C# 文档

    修改代码中，我们遵守以下编码样式：

    • C# 标识符命名规则和约定
    • 常见 C# 代码约定

    可以选择性阅读的教程：

    • C# 语言介绍：部分语言的细节，不需要全看。
    • 高级 .NET 编程文档 | Microsoft Learn：对C#和计算机基础知识感兴趣的话可以挑一些内容读一下。

    Unity基础介绍参考资料

    必看基础教程：

    • Unity引擎介绍
    • Learn Unity Beginner/Intermediate 2023 (FREE COMPLETE Course - Unity Tutorial)：来自CodeMonkey的免费教程，我们将大概仿照此项目制作。
    • Learn Unity Multiplayer (FREE Complete Course, Netcode for Game Objects Unity Tutorial 2023)：联网部分教程。

    Unity官方中文文档：

    • Unity C# Scripts APIs: C#脚本API文档，代码相关。
    • Unity User Manual 2022.3 (LTS)：编辑器相关。

    可以选择性阅读的教程：

    • Best practices for organizing your Unity project

    可能用得到的资源：

    • JKFrame
    • ET - Unity3D Client And C# Server Framework

    Unity作为一个对大部分用户都是免费的软件，相关的教程资料非常多。建议随时查找在线资料，并参考官方文档一同阅读。
